# Pomodoro Timer

A clean, minimal Pomodoro timer web app built with vanilla HTML, CSS, and JavaScript.

## Live Demo

[https://pomodoro-timer-six-ruby.vercel.app/](https://pomodoro-timer-six-ruby.vercel.app/)

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/Mansab019/pomodoro-timer.git

2. Navigate into the folder:
   cd pomodoro-timer

3. Open index.html with Live Server in VS Code
   (Right click index.html → Open with Live Server)

That's it. No installs, no build step, no dependencies.

## Features

- Configurable focus and break durations
- Auto-transitions between focus and break cycles
- Audio cue when a session ends
- Session history saved to localStorage
- History resets on a new calendar day
- Color theme changes between Focus and Break mode
- Fully responsive
